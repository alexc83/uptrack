# Dashboard UI Phase 1 Spec

## Overview

This is phase 1 of 2 for the dashboard UI layout. Use the screenshot referenced below for how it should look.

## Requirements for phase 1

- Generate PrimeNG components (sidebar and main content)
- Sidebar should always appear on the page
- Dashboard route at /dashboard (main content)
- Main dashboard layout and any global styles
- Create both dark and light modes. Reference screenshots below for both light and dark modes.
- Create light / dark mode button to the right of the app title at the top of the sidebar
- Icon to the left of the app title should have the color matching the screenshot with a similar icon
- Links on the sidebar for Dashboard, Credentials, and CE Records
- Bottom of the sidebar has a user avatar area and logout button as seen in the screenshots.
- Placeholder for main area. Just add an h2 with "Main" for now.
